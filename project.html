Operation of Two Switch Boosting Switched Capacitor Converters with PWM Control

The main aim of this project is to :

1. To convert a dc input voltage VS into a dc output voltage VO.

2. To regulate the dc output voltage against load and line variations.

3. By using different PWM techniques by adjusting duty cycle and frequency.
